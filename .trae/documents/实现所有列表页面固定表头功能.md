## 任务概述
为系统中所有的列表页面实现“表头固定”功能，以便在数据较多时，用户滚动查看列表内容时表头始终可见。

## 技术方案
在 Element UI 的 `el-table` 组件中，通过设置 `height` 或 `max-height` 属性即可实现固定表头。为了适配不同屏幕高度，我们将使用 CSS 的 `calc()` 函数动态计算表格高度。

## 实施步骤
1. **分析页面结构**：
   - 大部分列表页包含：系统顶栏、页面标题、搜索/筛选栏、表格卡片内边距、分页栏。
   - 经过估算，对于带搜索栏的页面，表格高度设为 `calc(100vh - 320px)` 较为合适。
   - 对于不带搜索栏的页面（如角色管理），高度设为 `calc(100vh - 240px)`。

2. **逐一修改以下页面文件**：
   - **司机管理**：`DriverList.vue`、`PhysicalManage.vue`
   - **车辆管理**：`VehicleList.vue`、`VehicleInsurance.vue`、`VehicleMaintenance.vue`、`VehicleInspection.vue`
   - **挂车管理**：`TrailerList.vue`、`TrailerInsurance.vue`、`TrailerMaintenance.vue`、`TrailerInspection.vue`
   - **系统管理**：`UserManage.vue`、`RoleManage.vue`

3. **具体改动示例**：
   ```html
   <el-table 
     :data="list" 
     v-loading="loading" 
     style="width: 100%"
     height="calc(100vh - 320px)"
   >
   ```

## 预期效果
- 所有列表页面的表头将固定在顶部。
- 表格内部会出现滚动条，而页面整体不再产生垂直滚动（或滚动范围极小），提升操作体验。
