## 任务概述
在保险记录（司机、车辆、挂车）的编辑和添加表单中，实现“购买日期”与“到期日期”的联动逻辑：当用户选择购买日期后，系统自动计算并设置到期日期为一年后的前一天。

## 实施步骤

### 1. 修改前端页面组件
针对以下三个页面进行修改：
- [DriverInsurance.vue](file:///c:/Users/10712/Documents/trae_projects/house/src/pages/drivers/DriverInsurance.vue)
- [VehicleInsurance.vue](file:///c:/Users/10712/Documents/trae_projects/house/src/pages/vehicles/VehicleInsurance.vue)
- [TrailerInsurance.vue](file:///c:/Users/10712/Documents/trae_projects/house/src/pages/trailers/TrailerInsurance.vue)

### 2. 实现联动逻辑
在每个组件的 `methods` 中添加日期计算方法：
- 监听“购买日期” (`form.start_date`) 的变化。
- 使用 `Date` 对象进行计算：
  1. 获取购买日期的年份并加 1。
  2. 将日期减去 1 天。
  3. 格式化为 `yyyy-MM-dd` 字符串。
- 将计算结果赋值给 `form.end_date`。

### 3. 绑定事件
在模板中的“购买日期” `el-date-picker` 组件上绑定 `@change="handleStartDateChange"` 事件。

## 技术细节
计算逻辑示例代码：
```javascript
handleStartDateChange(val) {
  if (val) {
    const date = new Date(val);
    date.setFullYear(date.getFullYear() + 1);
    date.setDate(date.getDate() - 1);
    this.form.end_date = date.toISOString().split('T')[0];
  }
}
```
